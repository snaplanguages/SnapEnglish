Unit 13: BUILD


Building Blocks of English
💡
1. Build around
Dựa trên ý tưởng/sự vật để phát triển
= base st on an idea or thing and develop it from there
"Successful businesses are built around good personal relationships."
💡
2. Build in
Thêm vào, gắn vào (cố định)
= make/include st as a permanent fixture / part of st else
"You could build in a wardrobe with mirrored doors."
"They have built an extra clause in the contract."
💡
3. Build on
Dựa trên cái gì / Xây thêm phòng
= base st on an idea / use achievements as a base / add more room
"Our relationship is built on trust."
"We built two new rooms on to make it larger."
💡
4. Build up
Gia tăng / Tâng bốc / Làm khỏe mạnh / Sáng tạo
= become greater / praise enthusiastically / make healthier / create
"You need more protein to build yourself up."
"She has built up a very successful business."
💡
5. Build over
Xây che kín lên cái khác
= construct st so that it extends over something else
"We built a deck out over the pond."
