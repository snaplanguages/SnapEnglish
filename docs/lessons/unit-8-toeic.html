1. Bị động trong các thì Đơn & Tiếp Diễn

🔥 Bị động các thì Đơn: S + to be + PII

  • Quá khứ: S + was/were + PII. [cite: 244]
  • Hiện tại: S + is/am/are + PII. [cite: 244]
  • Tương lai: S + will be + PII. [cite: 244]

🔥 Bị động các thì Tiếp Diễn: S + to be + being + PII

  • Quá khứ: S + was/were + being + PII. [cite: 246]
  • Hiện tại: S + is/am/are + being + PII. [cite: 246]
  • Tương lai: S + will be + being + PII. [cite: 246]

2. Bị động Hoàn Thành & Động từ Khuyết Thiếu

🔥 Bị động các thì Hoàn Thành: S + (had/have/will have) + been + PII

  • Quá khứ: S + had + been + PII. [cite: 248]
  • Hiện tại: S + have/has + been + PII. [cite: 248]
  • Tương lai: S + will have + been + PII. [cite: 248]

🔥 Bị động với Động từ khuyết thiếu

Cấu trúc: S + can/ could/ may/ should/ must + be + PII. [cite: 252]

💡 Mẹo nhớ: Câu bị động LUÔN LUÔN phải có To be và Phân từ II (PII/V3). Thiếu một trong hai là sai cấu trúc!
